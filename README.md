## About the Project

This is a **static landing page clone** inspired by the Cursor website.

What this project tries to replicate:
- Overall layout and design flow
- Hero section with CTA
- Brand trust section
- Feature explanation sections
- Reviews and testimonials
- Changelog and highlights
- Footer with multiple links

The project focuses more on **layout and styling**, not functionality.
